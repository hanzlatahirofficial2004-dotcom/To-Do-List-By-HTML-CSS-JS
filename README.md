# 📝 Dynamic Todo Application

A sleek, modern, and highly responsive **Todo Application** built using pure web technologies. This project focuses on efficient DOM manipulation, data structure management, and clean layout design using CSS Grid.

---

## ✨ Features
* **Grid-Based Layout:** Form inputs and task items are aligned perfectly using CSS Grid for a uniform, professional UI.
* **Smart Date Formatting:** Automatically converts standard inputs into a user-friendly `DD-MM-YYYY` format upon rendering.
* **Interactive Task Management:** Seamlessly add tasks using the "Add" button or by pressing the `Enter` key.
* **Instant State Updates:** Efficiently manages data state using JavaScript arrays and immediately re-renders the UI on adding or deleting items.
* **Input Validation:** Built-in safeguards to prevent adding empty tasks or incomplete dates.

---

## 📸 Core Preview & Logic
The app initializes with sample tasks tailored around key milestones in full-stack development, demonstrating clear lifecycle management from arrays to rendered HTML elements.

* **State Array:** `todos = [{ text, date }]`
* **Render Pipeline:** Clears the previous state, formats the date string, dynamically injects template literals, and appends rows to the container securely.

---

## 🛠️ Technical Stack
* **Structure:** HTML5 (Semantic Forms and Containers)
* **Styling:** CSS3 (CSS Grid, Hover Transitions, Box-sizing overrides, and Clean Typography)
* **Scripting:** Vanilla JavaScript (ES6+ array methods, event listeners, keyboard event capture, and dynamic innerHTML management)

---

## 🚀 How to Run this Project Locally
1. Clone this repository to your local machine.
2. Ensure all three files (`index.html`, `TO_DO_List.css`, and `TO_DO_List.js`) are located in the same root folder.
3. Open `index.html` in any web browser to run the application immediately.
